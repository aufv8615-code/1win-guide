# Affiliate review site (RU)

Это готовый сайт-обзор (Next.js) с внутренними страницами и SEO-базой.
Цель: собирать SEO-трафик по интентам (вход, регистрация, бонусы, выплаты, верификация, безопасность, FAQ) и вести на партнёрскую ссылку.

## Быстрый старт (самое простое)
1) Установи Node.js LTS (18+)
2) Открой папку проекта и выполни:
   - `npm install`
   - `npm run dev`
3) Открой в браузере: http://localhost:3000

## Деплой (рекомендую Vercel)
1) Создай аккаунт на Vercel
2) New Project → Import → выбери папку проекта
3) Deploy

## Что нужно поменять
Файл: `lib/site.js`
- `domain` — твой будущий домен (после покупки)
- `affiliateUrl` — уже стоит: https://lkhv.cc/885d69

## Важно
Сайт — независимый информационный обзор. 18+. Игра связана с риском финансовых потерь.
